# MMM producer golden fixture suite

`tests/fixtures/mip_export/golden_v1/index.json` is a versioned, deterministic
index over existing public producer-boundary fixtures. It catalogs success,
warning, failure, calibration rejection, extrapolation, Bayesian research-only,
and mixed-claim examples using relative paths only. Fixtures are examples, not
production authorization; they do not freeze the interface, authorize MIP
consumer readiness, recommendations, optimization, candidate plans, or response
surfaces. MIP parsing remains outside MMM.
